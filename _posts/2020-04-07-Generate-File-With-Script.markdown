# 用法 #
在主目录里运行genpost.py文件，根据提示输入即可

# 注意事项 #
1. 布局是检测_layouts文件夹里的文件然后提供选择
2. 标题只能是英文
3. 副标题和tags可以默认为空
4. tags可以输入多个，用逗号隔开
